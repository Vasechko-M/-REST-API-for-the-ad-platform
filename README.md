\# Readme.md



\## Описание проекта



Проект представляет собой RESTful API для управления пользователями, объявлениями и комментариями. Основные возможности:



\- регистрация и авторизация пользователей;

\- управление профилем (обновление данных, смена пароля, загрузка аватара);

\- работа с объявлениями: создание, просмотр, редактирование, удаление, загрузка изображений;

\- управление комментариями к объявлениям (добавление, редактирование, удаление);

\- получение информации о текущем пользователе и его объявлениях.

\## Используемые технологии и библиотеки



\- \*\*Язык программирования\*\*: Java.

\- \*\*Фреймворк\*\*: Spring Boot.

\- \*\*Валидация\*\*: Jakarta Bean Validation (JSR 380).

\- \*\*Работа с базой данных\*\*: Spring Data JPA / Hibernate.

\- \*\*Безопасность\*\*: Spring Security (аутентификация и авторизация).

\- \*\*Логирование\*\*: Logback / Log4j2.

\- \*\*Документация API\*\*: OpenAPI 3.0 (Swagger).



\## Структура API



\### Пользователи



| \*\*Метод\*\* | \*\*Путь\*\* | \*\*Описание\*\* |

|---|---|---|

| POST | /users/set\_password | Смена пароля пользователя |

| GET | /users/me | Получение информации о текущем пользователе |

| PATCH | /users/me | Обновление данных пользователя |

| PATCH | /users/me/image | Загрузка или обновление аватара |



\### Регистрация и авторизация



| \*\*Метод\*\* | \*\*Путь\*\* | \*\*Описание\*\* |

|---|---|---|

| POST | /register | Регистрация нового пользователя |

| POST | /login | Авторизация пользователя |



\### Объявления



| \*\*Метод\*\* | \*\*Путь\*\* | \*\*Описание\*\* |

|---|---|---|

| GET | /ads | Получение всех объявлений |

| POST | /ads | Создание нового объявления (с изображением) |

| GET | /ads/{id} | Получение информации об объявлении по ID |

| PATCH | /ads/{id} | Обновление объявления |

| DELETE | /ads/{id} | Удаление объявления |

| PATCH | /ads/{id}/image | Обновление изображения объявления |

| GET | /ads/me | Получение объявлений текущего пользователя |



\### Комментарии



| \*\*Метод\*\* | \*\*Путь\*\* | \*\*Описание\*\* |

|---|---|---|

| GET | /ads/{id}/comments | Получение комментариев к объявлению |

| POST | /ads/{id}/comments | Добавление комментария к объявлению |

| PATCH | /ads/{adId}/comments/{commentId} | Редактирование комментария |

| DELETE | /ads/{adId}/comments/{commentId} | Удаление комментария |



\## Модели данных



В проекте используются следующие основные сущности:



\- \*\*User\*\* — информация о пользователе.

\- \*\*Ad\*\* — объявление (товар/услуга).

\- \*\*Comment\*\* — комментарий к объявлению.



Для каждой сущности определены схемы валидации и структуры запросов/ответов согласно OpenAPI.



\## Требования к запуску



1\. Установите JDK 17 или новее.

2\. Используйте сборщик Maven или Gradle.

3\. Настройте параметры подключения к базе данных в application.properties.

4\. Для работы с API используйте авторизацию.



\## Примечания



\- Все эндпоинты, требующие авторизации, возвращают 401 Unauthorized при отсутствии или недействительной авторизации.

\- Валидация входных данных осуществляется на уровне моделей и контроллеров.

\- Для загрузки изображений используется multipart/form-data.



Проект разработан с учётом современных практик REST API и безопасности.
